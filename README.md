An accessible, **headless** radio group primitive for React. It handles selection
state, roving tabindex, full keyboard navigation, and ARIA — you bring the markup
and the styles. Works with **React 18 and 19**.

## Install

```bash
pnpm add headless-radio-group-react
```

> _Bleeding edge or before the first npm release: `pnpm add github:kea0811/headless-radio-group-react`._
